# Analysis_WechatSteps
用于从视频中提取微信步数汇总

使用方法

```
python analysis_forV.py
```

运行这份代码会从视频中提取步数

视频要求见项目的视频示例

偶尔误识别，错误率大概是3%吧，有时候会检测不到，只要找到是哪天没检测到，其他的往上推就可以，以后有空，或者真的有人用再更新吧

reshape是因为单位要求打印，放到word里看不清，最重要的就是上面说的analysis_forV.py，analysis.py用于对图片检测，其他的酌情使用吧
